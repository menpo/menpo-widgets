:mod:`menpowidgets.items`
=========================

This is the main function that can be used in order to visualize any Menpo
object including PointCloud, PointGraph, LabelledPointUndirectedGraph, TriMesh,
Image, MaskedImage, LandmarkManager and shape or appearance PCAModel. Note
that it does not support patches or patch-based PCAModel.
